## 📌Sobre o Projeto
> Este repositório contém o desenvolvimento de um sistema de análise de opurtunidades de crédito solicitado pela Faculdade de Tecnologia de São José dos Campos - Professor Jansen Vidal
>
> Este projeto foi desenvolvido em parceria com a DM com o objetivo de apoiar a análise de crédito a partir de dados econômicos do Brasil. A proposta é reunir, organizar e apresentar informações relevantes sobre concessão de crédito e nível de endividamento, facilitando a visualização e o entendimento desses indicadores. Com isso, busca-se oferecer uma base mais clara e acessível para auxiliar na tomada de decisão e na análise do cenário econômico.
> 
> Status do Projeto: Em andamento ⏳
<br>

## 📚 Metodologia
> O framework de Metodologia Ágil utilizado no produto foi o Scrum, um método ágil adaptativo, iterativo, flexível e eficaz. Entre as ferramentas utilizadas no Scrum, uma é a divisão do projeto em Sprints. Para selecionar quais seriam as entregas das nossas Sprints, primeiro definimos nosso MVP, priorizando as tarefas que trariam maior entrega de valor para o cliente. Então, a partir das Tarefas foi construído o Backlog do Produto,  o qual foi aprovado pelo cliente e dividido em 3 Backlog de Sprint.

> Dessa forma, com as Tarefas já traçadas, definimos a quantidade de tempo necessário para cada Tarefa, sendo dividido, de maneira mais otimizada, entre os Desenvolvedores do time.

## 🏆MVP
<img width="1920" height="1080" alt="MVP" src="https://github.com/user-attachments/assets/acc6b367-0fd8-45fa-a308-067cfa3c91f5" />


1️⃣ Sprint I: Dashboard funcional com dados reais > Coleta dados regionais do Banco Central, e expõe com ranking de estados e filtros por estado/período.

2️⃣ Sprint II: Inteligência analítica e alertas > Ranking de oportunidades por região com indicadores combinados.

3️⃣ Sprint III: Exportação, acesso e mobilidade > Geração de relatórios PDF/Excel exportáveis, sistema de login com perfis de acesso e interface responsiva para uso em dispositivos móveis.
